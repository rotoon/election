# Project Summary

## Project: Electric Pulsar (Online Election System)

- **Architecture**: Separated Frontend & Backend (Repository Pattern).
- **Frontend**: Next.js 14 (App Router), Tailwind, Shadcn/UI, Zustand.
- **Backend**: Node.js/Express (or Next.js API), Supabase (Database & Auth), Prisma.
- **Status**: Core features implemented (Admin, EC, Voter, Results).

## Completed Features

- Authentication (Login/Register)
- Admin: Constituency Management
- EC: Party & Candidate Management, Poll Control
- Voter: Voting & Change Vote
- Public: Results Dashboard
